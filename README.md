#  Task Manager

A simple full-stack Task Manager application built with **Node.js, Express, MongoDB, and Vanilla JavaScript**.

This project provides a RESTful API for managing tasks and a simple frontend interface to interact with it.

---

##  Features

- Create a task
- Get all tasks
- Get a single task
- Update a task
- Delete a task
- MongoDB database integration
- RESTful API structure

---

##  Tech Stack

### Backend

- Node.js
- Express
- MongoDB
- Mongoose

### Frontend

- HTML5
- CSS3
- Vanilla JavaScript
